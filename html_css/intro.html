"About Me"

I decided to take this class to become a more well-rounded computer scientist. I am well
acquainted with Java, passing both Programming I and II with A's as well as working under
professor Rachel Trana to create tutoring problems and research labs for Programming II
over the summer.In addition to my proficiency with Java, I also have a basis in graphic
design and this class will also help me build on my computer design portfolio for my
resume. I led a group of three in my Human Computer Interactions class to create a
mobile application that would give information about all the clubs at NEIU over the course
of the semester. I oversaw the design of the paper application, as well as the implementation
of the paper prototype on the website Marvel to add functionality and polish. In the words of
John Romero, co-founder of id Software, you might not think that programmers are artists,
but programming is an extremely creative profession. It's logic-based creativity. I would
love to find a job where I can put my creative side and coding passion to good use.

I would like to describe myself as a creative individual. I am knowledgeable in
Photoshop CS6, which I've used to create logos and designs for T-shirts. I am also in a Chicago-
based dance group that has participated in different events throughout the year, both in Chicago
and out of the state. I speak Spanish and English fluently. I also took Chinese and Japanese
during high school so I would like to get back to learning both languages again. My favorite
TV shows include The Office and Parks and Recreation.

The 6 XHTML Rules:

  • Elements must have an opening and closing tag.
  • Element names must be in lowercase.
  • End tags are required.
  • Empty elements are signified with a closing slash. Example: <br/>
  • Attribute values must be in quotation marks.
  • Attribute-value pairs must be written in full.

5 HTML Term Definitions

W3C
The World Wide Web Consortium, which is an international community that sets the
standards of the web to ensure its longevity.
Element
An individual component of an HTML document. Elements have start tags and end tags.
Attribute
Special words within an element's start tag that controls the element's behavior.
Browser
Application software that allows users to access and view html documents and web pages.
Responsive Design
A method of designing websites that makes them adaptable to a wide range of devices.
